What is HTML ?

  • HTML stands for Hyper Text Markup Language
  • It is a basic building block of website
  • It describes the structure of a Web page
  • It is skeleton of WebPage
  • It is not case sensitive
  • HTML elements tell the browser how to display the content

Explained

  • !DOCTYPE : defines that this document is an HTML5 document
  • html : It is the root element of an HTML page
  • head : It contains meta information about the HTML page
  • title : It specifies a title for the HTML page
  • body : It element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • h1 : It defines a large heading
  • h6 : Itdefines a large heading
  • p : element defines a paragraph

What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

tagname Hello World.(Content) /tagname

The HTML element is everything from the start tag to the end tag:

Tags in HTML

  • HTML tags are keywords which defines that how browser will format and display the content
  • Tags are enclosed within <> brackets
  • There are some tags which do not have any closing tag like br,hr ect.
  • A self closing tag in HTML does not need to be closed manually

HTML Attributes

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

Some Attributes

  • href : The tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to
  • src : The tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:
  • width and height : specify the width and height of the content (in pixels)
  • alt :The required alt attribute for the tag specifies an alternate text for an image
  • title : defines extra info about an element
  • style : The style attribute is used to add styles to an element, such as color, font, size, and more.

HTML Quotation

  • The HTML blockquote element defines a section that is quoted from another source. Browsers usually indent blockquote elements.
  • q :tag defines a short quotation.
  • abbr : tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP" "ATM".
    Marking abbreviations can give useful information to browsers, translation systems and search-engines.
  • address :tag defines the contact information for the author/owner of a document or an article.
    The contact information can be an email address, URL, physical address, phone number, social media handle, etc. The text in the address
    element usually renders in italic, and browsers will always add a line break before and after the
    element.
  • cite :tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.)

HTML Images

Images can improve the design and the appearance of a web page

The HTML img tag is used to embed an image in a web page. The img tag has two required attributes:
  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image

HTML Lists

HTML lists allow web developers to group a set of related items in lists.

Types of lists

  • Ordered List
  • Unordered list
  • Description list

Ordered List

An ordered list starts with the ol tag. Each list item starts with the li tag. The list items will be marked with numbers by default:
  1. Coffee
  2. Tea
  3. Milk

Unordered List

An unordered list starts with the ul tag. Each list item starts with the li tag. The list items will be marked with bullets (small black circles) by default:
  • Coffee
  • Tea
  • Milk

Description List

A description list is a list of terms, with a description of each term. The dl tag defines the description list, the dt tag defines the term (name), and the dd tag describes each term:
Coffee
- black hot drink
Milk
- white cold drink

HTML Span Element

The <span> element is an inline container used to mark up a part of text , or a part of document.

The <span> Element has no required attributes, but style, class and id are common.

Ex -: <p> I am <span style="color:green;"> green </span></p>

Output -: I am green
